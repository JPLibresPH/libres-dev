Hiyas Press Inc. Printing Management System

About

As the main front-end developer for this project, I incorporated key elements like the company’s color palette to match and brand the system according to their identity. I also explored the possibility of applying aesthetic UI design to this kind of system. This is also where I got introduced to a different CSS framework, Tailwind. It was kind of convenient to use, especially compared to Bootstrap if you’re a beginner in design—it allows for more flexibility when styling the system’s CSS.

Even though we went through a rough road and a roller-coaster journey, we were able to defend it—and thank God, we finished it.
